## 1. `echo` ‚Äì Print text or variables to the terminal

**Syntax:**

```bash

```

**Example:**

```bash

```

---

## 2. `pwd` ‚Äì Show present working directory

**Syntax:**

```bash

```

**Example:**

```bash

```

---

## 3. `whoami` ‚Äì Show current logged-in username

**Syntax:**

```bash

```

**Example:**

```bash

```

---

## 4. `ls` ‚Äì List files and directories

**Syntax:**

```bash
ls [options]
```

**Examples:**

```bash

ls -l
```

---

## 5. `touch` ‚Äì Create a new empty file

**Syntax:**

```bash
touch filename
```

**Example:**

```bash

```

---

## 6. `mkdir` ‚Äì Create a new directory

**Syntax:**

```bash
mkdir foldername
```

**Example:**

```bash

```

---

## 7. `cp` ‚Äì Copy files or directories

**Syntax:**

```bash
cp source destination
```

**Examples:**

```bash
cp file1.txt file2.txt
cp -r dir1/ dir2/
```

---

## 8. `mv` ‚Äì Move or rename files/directories

**Syntax:**

```bash
mv source destination
```

**Examples:**

```bash
mv file.txt folder/
mv oldname.txt newname.txt
```

---

## 9. `rm` ‚Äì Delete files or directories

**Syntax:**

```bash
rm filename
```

**Examples:**

```bash
rm file.txt
rm -r folder/
```

---

## 10. `rmdir` ‚Äì Remove empty directories

**Syntax:**

```bash
rmdir foldername
```

**Example:**

```bash

```
